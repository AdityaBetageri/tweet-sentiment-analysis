 # Tweet Sentiment Analysis using Machine Learning

## 📌 Project Overview

This project performs **sentiment analysis on tweets** using Machine Learning techniques and OpenAI embeddings.
The model analyzes tweet text and predicts whether the sentiment is **Positive, Negative, or Neutral**.

This project was developed as part of a **B.Tech Machine Learning project**.

---

## 🚀 Features

* Tweet text preprocessing
* Sentiment classification using machine learning
* OpenAI embeddings for text representation
* Custom tweet sentiment prediction
* Jupyter Notebook implementation

---

## 🛠️ Technologies Used

* Python
* Jupyter Notebook
* OpenAI Embeddings
* Scikit-learn
* Pandas
* NumPy

## ⚙️ How to Run the Project

1. Clone the repository

git clone https://github.com/AdityaBetageri/tweet-sentiment-analysis.git

2. Install required libraries

pip install pandas numpy scikit-learn openai

3. Open the notebook

Open `_Tweet_Sentiment_Project 2.ipynb` in **Jupyter Notebook** or **Google Colab**.
